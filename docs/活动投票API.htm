post 
http://127.0.0.1:8080/admin/create

Header

字段 类型 描述
token String

标记用户登录的token

参数

字段 类型 描述
title String

投票名称 (Not Null)
type Number

问题类型 1单选，2多选，3打分（十分制）,4 打分（百分制） (Not Null)
startTime Number

开始时间，毫秒级时间戳 (Not Null)
endTime Number

结束时间，毫秒级时间戳 (Not Null)
max Number

最多选择项
options Array

String型的选项数组
  • Request-Example: 
{
    "options": [
        "刘明",
        "涛涛",
        "俊俊",
        "老板"
    ],
    "title": "不洗碗工作室最佳程序员选举",
    "startTime": 1490427181,
    "endTime": 1490859181,
    "type": 1,
    "max": 4
}

Data

字段 类型 描述
vote Vote

返回voteResponse对象，包含一个投票的基本信息以及该投票下的选项
  options Array

nextPage Number

navigateFirstPage Number
